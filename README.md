# InfoFácil

App Android em Kotlin com Jetpack Compose para alfabetização digital básica.

## Stack
- Kotlin
- Jetpack Compose
- Material 3
- DataStore para progresso local offline

## Package
`com.info85.infofacil`

## Rodar o projeto
```bash
./gradlew assembleDebug
```

## Estrutura inicial
- `app/src/main/java/com/info85/infofacil/MainActivity.kt`
- `app/src/main/java/com/info85/infofacil/ui/InfoFacilApp.kt`
- `app/src/main/java/com/info85/infofacil/data/LocalModulesDataSource.kt`
- `app/src/main/res/drawable/infofacil_logo_foreground.xml`
